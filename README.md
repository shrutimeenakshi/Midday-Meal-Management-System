🍱 Midday Meal Management System

A smart web-based application developed to automate and monitor midday meal distribution in schools using QR-based attendance tracking, analytics, and health monitoring features.

The system improves transparency, reduces manual errors, and enables efficient management of meal distribution and student attendance through a centralized digital platform.

---

🚀 Features

✅ QR-based student attendance and meal verification
✅ Real-time meal distribution tracking
✅ Role-based admin and staff dashboards
✅ Student record management
✅ BMI calculation and health deficiency detection
✅ Nutrition recommendation system
✅ Analytics and reporting dashboard
✅ Secure database integration

---

⚙️ Installation

Clone the Repository

git clone https://github.com/shrutimeenakshi/Midday-Meal-Management-System.git

Navigate to Project Folder

cd Midday-Meal-Management-System

Install Dependencies

pip install -r requirements.txt

Run the Application

python app.py

Open in Browser

http://127.0.0.1:5000/
